aspire.exceptions.handle_exception(exc_type, exc_value, exc_traceback)¶

Handle any top-level unhandled exception.

Parameters

  • exc_type – Exception type object

  • exc_value – Exception value object (an instance of type exc_type)

  • exc_traceback – The Traceback object associated with exc_value (also available as exc_value.__traceback__)

Returns

On return, useful diagnostic information has been logged, and the exception re-raised.
