aspire.aspire.utils.array_utils module¶

This module contains functions useful for common tasks such as crop/mask/downsample etc. It is different from the functions in PreProcessor as they allow you to crop/downsample to a non square size. aspire.aspire.utils.array_utils.radius_norm(n: int, origin=None)¶

Create an n(1) x n(2) array where the value at (x,y) is the distance from the origin, normalized such that a distance equal to the width or height of the array = 1. This is the appropriate function to define frequencies for the fft of a rectangular image.

For a square array of size n (or [n n]) the following is true: RadiusNorm(n) = Radius(n)/n. The org argument is optional, in which case the FFT center is used.

Theta is the angle in radians.

(Transalted from Matlab RadiusNorm.m)
