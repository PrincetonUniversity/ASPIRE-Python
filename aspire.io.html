aspire.io.starfile.save_star(image_source, starfile_filepath, batch_size=1024, save_mode=None, overwrite=False)¶

Save an ImageSource to a STAR file + individual .mrcs files Note that .mrcs files are saved at the same location as the STAR file.

Parameters

  • image_source – The ImageSource object to save

  • starfile_filepath – Path to STAR file where we want to save image_source

  • batch_size – Batch size of images to query from the ImageSource object. Every batch_size rows, entries are written to STAR file, and the .mrcs files saved.

  • save_mode – Whether to save all images in a single or multiple files in batch size.

  • overwrite – Whether to overwrite any .mrcs files found at the target location.

Returns

None
