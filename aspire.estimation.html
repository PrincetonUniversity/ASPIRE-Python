get_covar(coeffs, ctf_fb=None, ctf_idx=None, mean_coeff=None, do_refl=True, noise_var=1, covar_est_opt=None)¶

Calculate the covariance matrix from the expansion coefficients and CTF information.

Parameters

  • coeffs – A coefficient vector (or an array of coefficient vectors) to be calculated.

  • ctf_fb – The CFT functions in the FB expansion.

  • ctf_idx – An array of the CFT function indices for all 2D images.

  • mean_coeff – The mean value vector from all images.

  • noise_var – The estimated variance of noise.

  • covar_est_opt – The optimization parameter list for obtaining the Cov2D matrix.

Returns

The basis coefficients of the covariance matrix in the form of cell array representing a block diagonal matrix. These block diagonal matrices may be manipulated using the blk_diag_* functions. The covariance is calculated from the images represented by the coeffs array, along with all possible rotations and reflections. As a result, the computed covariance matrix is invariant to both reflection and rotation. The effect of the filters in ctf_fb are accounted for and inverted to yield a covariance estimate of the unfiltered images.

Estimate the expansion coefficients using the Covariance Wiener Filtering (CWF) method.

Parameters

  • coeffs – A coefficient vector (or an array of coefficient vectors) to be calculated.

  • ctf_fb – The CFT functions in the FB expansion.

  • ctf_idx – An array of the CFT function indices for all 2D images.

  • mean_coeff – The mean value vector from all images.

  • covar_coeff – The block diagonal covariance matrix of the clean coefficients represented by a cell array.

  • noise_var – The estimated variance of noise.

Returns

The estimated coefficients of the unfiltered images in certain math basis. These are obtained using a Wiener filter with the specified covariance for the clean images and white noise of variance noise_var for the noise.

n = None¶

An object representing a 2*L-by-2*L-by-2*L array containing the non-centered Fourier transform of the mean least-squares estimator kernel. Convolving a volume with this kernel is equal to projecting and backproject-ing that volume in each of the projection directions (with the appropriate amplitude multipliers and CTFs) and averaging over the whole dataset. Note that this is a non-centered Fourier transform, so the zero frequency is found at index 1.
